# Study TypeScript

자바스크립트가 탄생한지 20년이 넘도록 규모와 수준은 나날이 발전하고 있는 반면에 Javascript 언어에 대한 발전은 제자리 걸음을 이어오고 있었습니다. 그래서 이를 보완하기 위한 것이 바로 TypeScript 입니다.   

오늘날 TypeScript는 다양한 환경에서 사용되고 있습니다. 

> TypeScript를 사용하는 대표적인 프론트엔드 환경
- 리액트 (React)
- 뷰 (Vue.js)
- 앵귤러 (Angular)
